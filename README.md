# Echo Translate

## Introduction
**Echo Translate: An Advanced Web Reading Assistant for Language Learners.** Echo Translate leverages advanced AI technology to provide you with fluid and intuitive word explanations and sentence translations. This tool is designed to simplify and enhance the language learning process, aiding users in effectively acquiring new language skills.

## Features
- **Instant Translation**: Translate text on web pages either by right-clicking and selecting **Echo Translate** or simply by double-clicking the text.
- **Multiple Languages Support**: Offers a wide range of languages to meet diverse learning requirements.
- **User-Friendly Interface**: Designed with user experience in mind, ensuring ease of use.
- **Customizable Settings**: Adapt the translation settings to fit your personal learning style in the extension's options page.

## Installation
To install **Echo Translate**, follow these steps:
1. Visit the Chrome Web Store.
2. Search for **Echo Translate**.
3. Click 'Add to Chrome' to install the extension.

## How to Use
- **Right-Click Translation**:
  1. Highlight the text you want to translate.
  2. Right-click and select **Echo Translate** from the context menu.
  3. The translation will appear in a popup.
- **Double-Click Translation**:
  1. Enable double-click translation in the settings page.
  2. Simply double-click on any text to see its translation.
